device-capabilities.js
A JavaScript library that detects the client device's capabilities for support of several sensors like touch, motion and orientation sesnors and also provides out of the box support for browser detection and OS/Platform detection.

Device detection
Detecting the type of device your site has been loaded in is now made easy. Attract visitors by making your site respond to different type of devices
Browser detection
Finding out which browser the user accesses your site is a piece of cake with this library. Carter users with information based on their favourite browsers
Platform detection
Identifing the platform the device is running is made simple.You can quicly Identify the platform/os the device is running to optimize your site for different platforms.
Feature detection
Checking wether the user's device and browser supports device specific API is a breeze. Impress your users by displaying the device's specific abilities.
